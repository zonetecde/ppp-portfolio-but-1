Présentation de la SAÉ 1.02

Durant cette SAÉ, nous avons comparer expérimentalement l'efficacité de deux types d'implémentation de listes triées d'objets pour les opérations d'ajout, de suppression et de test d'appartenance. Nos principales réalisations incluent :

  • ✔️ Une implémentation d'une liste contiguë
  • ✔️ Une implémentation d'une liste chainée
  • ✔️ Une implémentation optionnelle d'une liste chainée utilisant une liste chainée pour gérer ses places libres
  • ✔️ Une génération automatique de fichier CSV pour les résultats en fonction du nombre d'éléments de base dans la liste
  • ✔️ Une génération automatique de graphique à partir des fichiers CSV avec trois courbes pour comparer les trois implémentations
  • ✔️ Une écriture en pseudo-code des algorithmes utilisés

Les thèmes abordés incluent le développement Java, la conception d'un jeu, l'interface utilisateur et la communication entre deux machines.
